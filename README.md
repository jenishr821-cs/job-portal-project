## 📖 About

**ZentruaFlask** is a modern online gaming store developed using **Python Flask** and **MySQL**. It provides users with a seamless experience to browse games, purchase titles, manage profiles, and explore trending or mystery games through a clean, responsive interface.

This project was developed for learning full-stack web development while implementing real-world authentication, database management, and e-commerce concepts.

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ZentruaFlask.git
```

Move into the project folder:

```bash
cd ZentruaFlask
```

---

## 2️⃣ Create a Virtual Environment

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Import the Database

Import the SQL file:

```text
DATABASE/zentruadb.sql
```

into your MySQL server.

---

## 5️⃣ Configure Database

Open:

```text
config.py
```

Update your MySQL credentials.

Example:

```python
MYSQL_HOST = "localhost"
MYSQL_USER = "root"
MYSQL_PASSWORD = ""
MYSQL_DB = "zentruadb"
```

---

## 6️⃣ Run the Application

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
